Etiquetas HTML básicas

Las etiquetas HTML son la base principal del lenguaje que se usa para elaborar páginas web y que entienden los navegadores para construir la página correctamente y mostrarla al usuario a través de éste. En esta entrada te enseñaré las etiquetas básicas que deberías conocer por si necesitas hacer alguna modificación en tu página web, o por si al contrario te has animado a tocar el código y has hecho algún desajuste.
